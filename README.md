## Image Viewer

This simple application displays images from a directory. It showcases several Electron APIs.

Electron APIs: `app`, `BrowserWindow`, `Menu`, `dialog`, `shell`, `protocol`, `ipcRenderer`, `ipcMain`
NodeJS: `fs`, `path`, `util`,
